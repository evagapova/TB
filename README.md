# Задачи по теории вероятностей
## Тема 1. Моделирование геометрической вероятности
Условие задачи:
Загадываются два действительных числа от 0 до 4.
Какова вероятность, что их сумма больше 3?

Теоретическое решение:

А - сумма чисел x и y больше 3; Р(А) - вероятность того, что x + y > 3

$P(A) = \frac{m}{n} = \frac{16 - \frac{1}{2} \cdot 3^2}{16} = \frac{11.5}{16} = 0.719$
